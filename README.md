# arcbrimexotics.com

Arcbrim Exotics website. Built with Astro. Hosted on Hostinger.

## Stack

- **Framework**: Astro 4
- **Fonts**: Cormorant Garamond + Jost via Google Fonts
- **Hosting**: Hostinger (static output)
- **Source control**: GitHub

## Local Development

```bash
npm install
npm run dev
```

## Deploy

```bash
chmod +x deploy.sh
./deploy.sh
```
